Bem Vindo Ao Site de Licenciatura em CIG 2023

Site criado em forma de aprendizado de programaçao em Java Script, com isso é um site Noob. a ideia era criar de forma criativa um site que armazene informaçoes gerais da turma onde conteria diversas funcionalidades de auxilio e incentivo, como raking de diversos status dos dos estudantes da turma, como tambem o bate-papo, para uma possivel interraçao dentro do site, nao alcaçamos as espectativas ainda, mas estamos trabalhando para o site estar a 100%.


Este Site foi desenvolvido usando linguagem C++ Em Java Script no visual studio code foram adotadas varias tecnicas de programaçao interretiva, gozando e vislumbrando o Maximo de funçoes, manipulando o DOM e tentando ir de acordo com o agradavel aos olhos manipulando CSS5!

Esperamos que gostem e desfrutem!
